# Dynamic Team Professional Page

## Professional Profile

This is a professional profile web page project. 
It contains two professional profiles; when a click triggers the second profile, javaScript will hide the first profile and shows the second one. 
The page was built entirely with HTML, CSS and JavaScript to create a dynamic show/hide web page. 

# Installation

- Installation is not needed.

## Usage
You may:

- Create dynamic team page to your website.
